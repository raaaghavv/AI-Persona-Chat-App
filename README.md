## ğŸš€ Installation

Clone the repo:

```bash
git clone https://github.com/raaaghavv/ai-persona-chat.git
cd ai-persona-chat
```

Install dependencies:

```bash
npm install
```

Set up API keys in .env (choose one or both):

```bash
# For OpenAI
export OPENAI_API_KEY=your_key_here

# For Gemini
export GEMINI_API_KEY=your_key_here
```

Run the dev server:

```bash
npm run dev
```
